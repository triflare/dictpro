Bring the power of JSON objects & arrays to your TurboWarp projects! A more advanced, power user-oriented version of the vanilla Dictionaries extension from the TurboWarp Extension Gallery.

Unlike the original, you now have things like nested objects, array-only dictionaries, and more.

## Primary features

- **Based on Mint:** Dictionaries Pro's source code uses the Mint tooling, meaning it'll be rock-hard and easy to maintain in the future.
- **Advanced queries & array operations:** Powerful blocks for filtering arrays, aggregations (sum/average/min/max), deep searching, flattening, and Base64 export — plus full array helpers (initialize, push, get/set/insert/remove, join).
- **Nested path access & safe JSON handling:** Dot-notation path access (with escaping), automatic creation of nested structures, JSON import/export, and sanitization to protect against unsafe keys (e.g., `__proto__`).
